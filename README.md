# Astrophysics Programming

Course materials and projects for "Advanced Programming for Astrophysics", an academic course offered at the University of Trieste (UniTS).

> [!WARNING]
> Since this repository contains submodules, please clone using the following command:
> 
> ```bash
> git clone --recurse-submodules https://github.com/Spina02/Astrophysics-Programming
> ```
> 
> Or, after cloning normally, run:
> 
> ```bash
> git submodule update --init --recursive
> ```

## Repository Structure

- **[exercises](https://github.com/Spina02/Astro-Lab)**: 10 programming exercises covering fundamental algorithms and numerical methods

- **projects**: 3 project submodules:
    - **[solar-system](https://github.com/Spina02/solar-system)**: 2D N-body simulation
    - **[particle-mesh](https://github.com/Spina02/particle-mesh)**: particle-mesh simulation for astrophysical systems
    - **[1d-sph](https://github.com/Spina02/1D-SPH)**: 1D Smoothed Particle Hydrodynamics simulation

Each submodule contains detailed documentation on compilation and execution instructions.
